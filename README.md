# Комплекс ФСИН: Сквозное моделирование

Программный комплекс для расчета, визуализации и физического воплощения антисимметричного сфирального резонанса. Система демонстрирует бесшовный переход от математического сигнала к 3D-геометрии проводника и биоморфной кинематике робота.

## Архитектура системы

Проект реализует концепцию сквозного моделирования, состоящую из трех взаимосвязанных уровней:

| Уровень | Модуль | Функция | Индикатор корректности |
| --- | --- | --- | --- |
| **I. Энергия** | Нейросеть (Матрица ФСИН) | Генерация и балансировка фазовых сигналов. | Индекс устойчивости $\approx 0$. |
| **II. Форма** | 3D-Сфираль (S-узел) | Пространственная проекция пути тока без углов. | Плавность свития, отсутствие разрывов. |
| **III. Жизнь** | Биоморфный Гексапод | Тестирование сигналов на сложной механике. | Плавная походка без вибрации центра масс. |

## Математическое ядро

### 1. Золотой вурф и константы

Все пропорции системы масштабируются единым коэффициентом — Золотым вурфом (**1.309**). Габариты базовых структур привязаны к стандарту Казенной сажени (**217.6** мм).

* Длина корпуса гексапода: **217.6** мм.
* Соотношение сегментов ноги (Таз : Бедро : Голень): **1 : 1.309 : 1.713** (где $1.713 = 1.309^2$).

### 2. S-переход (Функция активации)

Генерация управляющих сигналов узлов производится по формуле антисимметричного фазового перехода:

$$f(x) = \tanh(\sin(\omega t + \phi) - \sin(\omega t - \phi))$$

* $\omega = 1.236$ (базовая частота шага/резонанса).
* $\phi$ — фазовый сдвиг, определяемый позицией узла в сети.

При четном количестве узлов достигается баланс: сумма векторов в любой момент времени $t$ равна нулю. Это обеспечивает компенсацию инерционных моментов (нулевую вибрацию).

## Интеграция в аппаратную часть (Hardware)

Сигналы Уровня I предназначены для управления физическим роботом-гексаподом.

**Спецификация аппаратной части (BOM):**

* **Контроллер:** ESP32 или Arduino Mega.
* **Приводы:** 6 сервоприводов (рекомендуются шаговые двигатели или coreless-серво для минимизации люфта).
* **Балансировка:** 4 внутренних гироскопических маховика (для компенсации веса моторов и аккумулятора в 10-узловой сети).
* **Рама:** 3D-печатная конструкция (длина **217.6** мм, ширина **166.2** мм).

## Алгоритм прошивки (C++ для ESP32/Arduino)

Код транслирует матрицу состояний ФСИН в ШИМ-сигналы для сервоприводов.

```cpp
#include <Servo.h>
#include <math.h>

// Константы ФСИН
const float OMEGA = 1.236;
const float WURF = 1.309;
const int NODES = 6; 

// Пины сервоприводов (Бедро)
const int SERVO_PINS[NODES] = {2, 3, 4, 5, 6, 7};
Servo legs[NODES];

float t = 0.0;
const float TIME_STEP = 0.05;

void setup() {
  for(int i = 0; i < NODES; i++) {
    legs[i].attach(SERVO_PINS[i]);
  }
}

// Вычисление S-перехода
float calculateSignal(float current_t, float phase_shift) {
  float f_loop = sin(OMEGA * current_t + phase_shift);
  float b_loop = -sin(OMEGA * current_t - phase_shift);
  return tanh(f_loop + b_loop);
}

void loop() {
  for(int i = 0; i < NODES; i++) {
    // Фазовый сдвиг (левая/правая сторона)
    float phase = (i >= NODES / 2) ? PI : 0; 
    
    // Получение сигнала от -1.0 до 1.0
    float signal = calculateSignal(t, phase);
    
    // Масштабирование сигнала в угол поворота сервопривода (90 +- 45 градусов)
    // Умножение на Wurf задает амплитуду шага
    int angle = 90 + (signal * 45 * WURF);
    
    // Ограничение углов для безопасности механики
    angle = constrain(angle, 0, 180);
    
    legs[i].write(angle);
  }
  
  t += TIME_STEP;
  delay(20); 
}

```

## Требования и установка

Требуется Python 3.10+.

1. Клонировать репозиторий:

```bash
git clone https://github.com/ВАШ_ЛОГИН/fsin-complex.git
cd fsin-complex

```

2. Установить зависимости:

```bash
pip install -r requirements.txt

```

3. Запустить интерактивную среду:

```bash
streamlit run app.py

```

## Структура репозитория

* `app.py` — главный файл приложения (интерфейс, расчеты, 3D-визуализация).
* `requirements.txt` — список зависимостей.
* `notebooks/analysis.ipynb` — Jupyter Notebook для анализа сгенерированных генетических матриц (`.csv`).
