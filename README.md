# FSIN-Simulator: Fractal Sfiral Artificial Neurons

Программный комплекс для моделирования и анализа фрактальных сфиральных искусственных нейронов (ФСИН). Проект объединяет принципы биоморфного машиноведения, проективной геометрии и теории активного времени. 
Инструментальная среда для моделирования и анализа фрактальных сфиральных нейронов. Позволяет верифицировать динамическую устойчивость сетей, построенных на принципах зеркальной антисимметрии и биоморфного пропорционирования по С.В. Петухову (Золотой вурф 1,309).

## Теоретический базис
В основе модели лежит синтез трех концепций:
1. **Золотой вурф (1,309):** Инвариант проективной геометрии ($\Phi^2/2$), определяющий кинематические пропорции живых систем. Применяется для нормирования весовых коэффициентов сети.
2. **Русская механика:** Система парных мер (саженей), обеспечивающая объемную соразмерность и подавление резонансов.
3. **Сфиральная топология:** Зеркальная антисимметрия и S-образный фазовый переход, реализующие эффект самокомпенсации сил и исключающие накопление ошибки.

## Функциональные возможности
- Визуализация динамики фазовых переходов в многослойных сетях.
- Интерактивная настройка биоморфных весов (Золотой вурф).
- Экспорт матриц состояний в формат CSV для глубокого анализа.
- Модуль статистической проверки индекса структурной устойчивости.
