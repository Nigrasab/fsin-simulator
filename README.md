# fsin-simulator
Fractal Sfiral Artificial Neurons Simulator
Назначение приложенияРазвернутое веб-приложение — это интерактивный симулятор фрактальных сфиральных искусственных нейронов (ФСИН).Задачи симулятора:Визуализация баланса сил: Демонстрация когерентного поведения сети при прохождении многомерного сигнала. Графики доказывают отсутствие экспоненциального роста амплитуды (перегрузки сети) благодаря S-образным фазовым переходам.Верификация биоморфного ядра: Подтверждение гипотезы о том, что весовые коэффициенты, отмасштабированные по пропорции золотого вурфа ($1.309$), обеспечивают структурную устойчивость вычислительной модели, подобно кинематике живых организмов.Генерация датасетов: Формирование числовых матриц (состояние каждого узла в конкретный такт времени) для независимого математического анализа.Интеграция функции экспорта трансформирует приложение из инструмента визуальной демонстрации в полноценный научно-исследовательский комплекс. Полученный CSV-файл позволяет загружать результаты эмуляции в среды квантовых вычислений (Qiskit) или сторонние аналитические программы (MATLAB, Jupyter) для вычисления метрик энергоэффективности алгоритма.
